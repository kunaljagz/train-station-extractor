# Train Station Extractor

## Setup Instructions
1. Clone the repository:
   
   ```bash
   git clone https://github.com/kunaljagz/train-station-extractor.git
   ```

2. Navigate into the project directory:
   
   ```bash
   cd train-station-extractor
   ```

3. Install the required dependencies:
   
   ```bash
   pip install -r requirements.txt
   ```
    
## Features Overview
- **Efficient Data Extraction:** Extracts train station information from various data sources.
- **Multiple Format Support:** Supports extraction from CSV, JSON, and XML.
- **Customizable Queries:** Allows users to define custom queries for specific data extraction needs.

## Usage Guide
1. To run the extractor, use the following command:
   
   ```bash
   python extractor.py --input <input_file> --output <output_file>
   ```

2. Replace `<input_file>` with the path to your input file and `<output_file>` with the desired output file path.

3. For more options, run:
   
   ```bash
   python extractor.py --help
   ```
